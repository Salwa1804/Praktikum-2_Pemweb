1. Struktur Dasar Halaman

A. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web modern?

CSS dan JavaScript digunakan dalam pemrograman web modern karena CSS berfungsi untuk mengatur tampilan visual, seperti warna, ukuran teks, tata letak, dan jarak antar elemen, sehingga halaman web menjadi lebih rapi dan enak dilihat. Sedangkan JavaScript digunakan untuk menambahkan interaktivitas dan perilaku dinamis, misalnya menampilkan pesan, menyembunyikan atau menampilkan elemen, mengubah mode tampilan, dan merespons aksi pengguna tanpa harus memuat ulang halaman.

B. Jelaskan konsep dan tujuan utama dari Responsive Web Design (RWD)!

RWD adalah desain web yang bertujuan membuat tampilan sebuah website secara otomatis terhadap berbagai ukuran layar, mulai dari desktop, tablet, hingga smartphone. RWD menggunakan teknik seperti layout fleksibel, grid responsif, dan media query agar elemen-elemen pada halaman dapat berubah ukuran, posisi, dan proporsinya sesuai perangkat yang digunakan. Tujuan utamanya adalah memastikan pengguna tetap mendapatkan pengalaman membaca dan navigasi yang nyaman tanpa harus melakukan zoom atau scroll.

C. Bagaimana kombinasi CSS, JavaScript, dan prinsip RWD dapat menghasilkan tampilan web yang interaktif sekaligus adaptif? Jelaskan bagaimana ketiganya saling menghadapi!

Kombinasi CSS, JavaScript, dan prinsip Responsive Web Design (RWD) dapat menghasilkan tampilan web yang interaktif sekaligus adaptif karena memiliki fungsi yang saling melengkapi. Yang dimana CSS berperan sebagai pengatur tampilan visual, mulai dari warna, ukuran teks, layout, hingga efek transisi. JavaScript kemudian menambahkan interaktivitas dengan memungkinkan elemen-elemen pada halaman, seperti tombol yang dapat mengubah mode tampilan, konten yang bisa disembunyikan, atau pesan konfirmasi yang muncul saat elemen diklik. Sedangkan, RWD memastikan seluruh elemen yang telah diatur oleh CSS dan dihidupkan oleh JavaScript tetap tampil dengan baik pada berbagai ukuran layar menggunakan layout fleksibel dan media query. Oleh karena itu, CSS memberikan desain visual, JavaScript memberikan perilaku dinamis, dan RWD memastikan keduanya tetap berfungsi secara optimal di semua perangkat.
